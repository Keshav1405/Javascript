Q1. Difference between “ == “ and “ === “ operators.

Double equals named as Equality Operator. " == " compares only the values of the two variables being compared.

Triple equals named as Identity / Strict equality Operator." === " compares both the values and data type of the two variables being compared.

Q2. What is the spread operator?

The spread operator(...) allows us to copy all or part of an existing array or object into a new array or object. The new array could be empty or have some values in it, the spread operator will concat the value of previous array into the new array.

Q3. What are the differences between var, let and const?

They are all keywords to declare a variable with some differences which are as follows:

  • var can be updated and redeclared. It's scope is functional scope and its default value without initialisation is 'undefined'.
  • let can be updated but can't be redeclared. It's scope is block scope and it cannot be accessed without initialisation.
  • const can neither be updated nor redeclared. It's scope is block scope and it cannot be accessed without initialisation.

Q4. What is execution context?

The execution context is used to describe how a code works inside. The specified code is parsed line by line during the execution context, and the variables and functions are saved in memory. The code is evaluated and then executed in an execution context, which is similar to a container that contains variables. First of all, a 'Global execution context' is created in which the global scope variables are stored and a 'function execution context' is created when there is a function call in the code.

Q6. What are closures? Give an example of closure.

When a function is declared inside another function then the child function can inherit the values from it's lexial parent function, this is known as closures in JS. Closures are created every time a function is created, at function creation time.

Q7. Explain call(), apply() and, bind() methods. Give an example of call(), apply(), bind().

  • Call is a method in function that helps us to replace the value of 'this' inside a function with whatever value we want. Some additional arguments can alse be given for the variables inside the function.
  • Apply is very similar to the call method. The only difference is that in apply we can pass an array as an argument list.
  • Bind is a method in function that helps us to create another function that we can execute later as per our requirement.
